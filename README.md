# python-icmp-tunnel

A Python implementation of ICMP Tunnel.

## Features

- [ ] 接受入站 TCP 连接
- [ ] 接受入站 UDP 连接
- [ ] 使用 Tunnel 接受整个操作系统的所有 IP 包
- [ ] ICMP Payload Encryption
- [ ] 借助 [KCP协议^1^][1] 降低通信延迟
